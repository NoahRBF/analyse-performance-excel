# Analyse de performance commerciale — Excel

## Contexte

Une entreprise souhaite analyser ses performances commerciales afin d’identifier les leviers d’amélioration pour l’année suivante.

## Objectif

Analyser les ventes afin de :
- Identifier les produits les plus performants
- Repérer les variations saisonnières
- Comparer les performances par région
- Proposer des recommandations stratégiques

## Données

Jeu de données comprenant :
- Date de vente
- Produit
- Région
- Chiffre d’affaires
- Quantité vendue

## Méthodologie

- Nettoyage et structuration des données
- Création d’indicateurs clés (KPIs)
- Tableaux croisés dynamiques
- Construction d’un dashboard synthétique

## Indicateurs clés analysés

- Chiffre d’affaires total
- Évolution mensuelle
- Top produits
- Performance par région
- Saisonnalité
